## 🎯 Usage

### Like a Wallpaper
Click the ♡ button to like a wallpaper. The heart will fill and count increases.

### Download Wallpapers
Click the ⬇ Download button to simulate a download.

### Search
Use the search bar to find wallpapers by keyword.

### Load More
Scroll to bottom and click "Load More Wallpapers" to see additional content.

### Filter by Category
Click filter tabs to view wallpapers by category.

## 🔧 Customization

### Change Colors
Edit the CSS variables in `style.css`:
```css
:root {
  --primary-color: #00d4ff;
  --secondary-color: #ff0080;
  --accent-color: #00ff88;
  /* ... */
}
```

### Add More Wallpapers
Edit `index.html` and add more `.wallpaper-card` divs in the `.wallpaper-grid`.

### Modify Layouts
Adjust grid columns in CSS:
```css
.wallpaper-grid {
  grid-template-columns: repeat(auto-fill, minmax(280px, 1fr));
}
```
